# スキル・得意領域

## クリエイティブ領域
- コピーライティング: TVCMからバナーまで。キャッチコピー、ボディコピー、スクリプト、コンセプトワーク全般
- クリエイティブディレクション: キャンペーン全体の設計・統括
- プランニング: コンセプト設計、MVV策定、ブランディング
- インタビュー・取材: 人から話を引き出し言語化する
- 映像企画: 台本・構成作成（篠原ともえ×TOKYO CREATIVE SALON映像企画など）
- 映画脚本: リモート映画の脚本執筆経験あり

## テクノロジー領域
- プロダクト開発: Claude Codeを使ったラピッドプロトタイピング
- フロントエンド: React / Next.js（Claude Code経由）
- デプロイ: Vercel
- API連携: Anthropic API, Replicate API, Web Audio API
- バージョン管理: GitHub

## 開発スタイル
- Claude Codeを活用した高速プロトタイピング
- コア機能ファースト→イテレーションで改善
- シンプルさ重視、機能の詰め込みを避ける

## 仕事の哲学
一番問題になっているところはなんだろう？ゴールは何？それを解決するための筋道をどう描いたらいいか、それを日常的に考えるのをクセにしている。それが見定められるととてもシンプルな形になって言語化できて気持ちよく伝えられるようになる。言葉を決めて提示するのが自分の役割。

## 仕事の範囲（継続案件の例）
- 企画とコピー
- 週1のMTG
- コピーを考える
- MVVなどまとめる
- 企画書を書く
- 各種相談
- 基本的には、話して、頭で考えて、スライドにまとめるところまでが領域
